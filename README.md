## Installation

### Dependencies

```bash
sudo add-apt-repository ppa:floe/libtisch
sudo apt-get update
sudo apt-get install libfreenect libfreenect-dev libfreenect-demos
```

Check if it is working by running:

```bash
freenect-glview
```

(source: <http://openkinect.org/wiki/Getting_Started>)
